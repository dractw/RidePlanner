# RidePlanner
Telegram bot for planning rides

_Under active development_

## Build
Create `.env` file with required variables ([example](https://github.com/dractw/RidePlanner/blob/main/.env.example))

Simply run `npm i`

# Run
Dev `npm run dev`
Prod `npm start` 

## Deploy
Create `.env` file with required variables ([example](https://github.com/dractw/RidePlanner/blob/main/.env.example))

Run `docker-compose up -d`
